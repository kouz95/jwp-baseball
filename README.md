# 숫자 야구 게임

## Events

- [ ] 게임 시작을 누른다.
  - [ ] 게임이 시작된다.
    - [ ] 목표 숫자가 설정된다.
- [ ] 사용자로 부터 숫자를 입력 받는다.
  - [ ] 입력에 대한 힌트를 제공한다.
  - [ ] 정답일 경우 종료.
- [ ] 게임 조회를 요청한다.
    - [ ] 사용자가 입력한 숫자와 힌트를 조회한다.
- [ ] 정답을 확인한다.
    - [ ] 목표 숫자를 출력한다.
- [ ] 재시작을 입력한다.
  - [ ] 게임이 재시작 된다.
- [ ] 종료를 입력한다.
  - [ ] 게임 종료 화면을 노출한다.

  
## 추가 요구 사항
- 사용자는 자신이 입력한 숫자들에 대한 힌트를 조회 할 수 있다.
